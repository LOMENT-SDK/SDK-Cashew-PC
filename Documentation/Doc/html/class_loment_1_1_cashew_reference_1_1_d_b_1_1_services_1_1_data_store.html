Detailed Description

The usage of AppConfig is simple. First, call DataStore.LoadAppConfig() (done). Next, when editing app wide default config settings, use MainViewModel.AppConfigViewModel, followed by AppConfigViewModel.UpdateAppConfigCommand.Execute() Finally, when composing a new Email, set the UI in accordance with the MainViewModel.AppConfigViewModel.Entity properties
