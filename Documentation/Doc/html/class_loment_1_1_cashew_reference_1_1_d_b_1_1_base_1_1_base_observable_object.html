void Loment.CashewReference.DB.Base.BaseObservableObject.VerifyPropertyName ( string  propertyName)

Verifies that a property name exists in this ViewModel. This method can be called before the property is used, for instance before calling RaisePropertyChanged. It avoids errors when a property name is changed but some places are missed.

This method is only active in DEBUG mode.

Parameters
propertyNameThe name of the property that will be checked.
