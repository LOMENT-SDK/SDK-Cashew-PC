Detailed Description

A interface that defines the necessary methods needed to provide an encryption engine used by the Cashew SDK. A custom encryption engine can be provided by an implementation of this interface. Sdk.Initialize(IDeviceInformationProvider, IRabbitMQService, IDataCrypter)
